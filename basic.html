PLC Basic

PLC를 처음 볼 때 알아야 하는 기초 개념

PLC를 처음 배우는 사람은 먼저 주소 체계와 기호를 읽을 수 있어야 한다. 이 페이지에서는 X, Y, M, T 같은 기본 주소와 a접점, b접점, 코일, 자기유지, 인터록의 개념을 쉽게 정리하였다.

PLC 주소 보는 법

제조사마다 약간 차이는 있지만, 입문할 때는 아래처럼 이해하면 된다.

X

입력

버튼, 스위치, 센서처럼 PLC로 들어오는 신호를 의미한다.

예: X0, X1, X2

Y

출력

램프, 부저, 밸브, 모터처럼 PLC가 내보내는 제어 신호를 의미한다.

예: Y0, Y1

M

내부 릴레이

PLC 내부에서만 사용하는 보조 기억 공간이다. 자기유지나 조건 기억에 많이 사용된다.

예: M0, M1

T

타이머

일정 시간이 지난 뒤 동작하게 하거나 지연 동작을 만들 때 사용한다.

예: T0, T1

C

카운터

입력된 횟수를 세는 기능이다. 제품 개수 세기 같은 작업에 사용된다.

예: C0, C1

D

데이터 레지스터

숫자 데이터나 설정값을 저장하는 공간이다. 입문 단계에서는 이름만 알아도 충분하다.

예: D0, D10

래더 회로 기호 읽는 법

회로를 직접 짜려면 기호를 읽을 수 있어야 한다. 아래 기호는 가장 많이 쓰는 기본 기호들이다.

NO Contact

a접점

[ X0 ]

평소에는 열려 있고, 입력이 들어오면 닫히는 접점이다. 버튼 ON, 센서 ON 같은 조건을 만들 때 많이 사용한다.

NC Contact

b접점

[ /X0 ]

평소에는 닫혀 있고, 입력이 들어오면 열리는 접점이다. 정지 조건, 반대 동작 금지 조건을 만들 때 자주 사용한다.

Coil

코일

( Y0 )

회로의 결과로 켜지는 출력이다. 램프, 밸브, 모터 출력에 연결된다.

Memory

내부 릴레이 코일

( M0 )

실제 출력이 아니라 PLC 내부에서 상태를 기억하기 위해 사용한다.

기본 회로 개념

회로를 직접 짜려면 아래 세 가지 개념을 먼저 이해하는 것이 중요하다.

Self Holding

자기유지

버튼을 한 번 눌렀을 때 손을 떼어도 동작이 계속 유지되도록 만드는 회로이다. 스타트 버튼 이후 계속 돌아가야 하는 장치에 많이 사용된다.

Interlock

인터록

두 개의 동작이 동시에 일어나면 안 될 때 서로를 막는 회로이다. 예를 들어 전진 중에는 후진 금지, 후진 중에는 전진 금지 같은 경우에 사용한다.

Timer

타이머

일정 시간이 지난 뒤에 동작하게 하거나, 잠시 지연 후 동작하게 만드는 회로이다. PLC다운 제어를 만들 때 자주 등장한다.

회로 짜는 순서

회로를 처음 짤 때는 아래 순서로 생각하면 훨씬 쉽다.

1. 입력 정하기

버튼, 센서, 스위치 중 어떤 신호가 들어오는지 먼저 정한다.

2. 출력 정하기

램프, 밸브, 모터 중 어떤 장치를 움직일지 정한다.

3. 조건 정하기

언제 켜지고 언제 꺼질지 논리 조건을 정한다.

4. 유지/해제 정하기

자기유지, 인터록, 센서 종료 조건이 필요한지 확인한다.
