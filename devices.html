Devices

입력 기기와 출력 기기

생산자동화기능사 실기에서는 입력 기기와 출력 기기를 먼저 구분할 수 있어야 한다. 입력은 PLC로 들어오는 신호이고, 출력은 PLC가 밖으로 내보내는 제어 신호이다.

Input

입력 기기

입력 기기는 버튼, 스위치, 센서처럼 PLC에 신호를 넣어주는 장치이다.

  • 푸시버튼
  • 리미트 스위치
  • 포토 센서
  • 유도 센서
  • 용량 센서
  • 초음파 센서

회로에서는 보통 X 주소를 사용해서 표현한다. 예: X0, X1, X2

Output

출력 기기

출력 기기는 PLC가 제어하는 장치로, 실제 동작을 수행한다.

  • 램프
  • 부저
  • 솔레노이드 밸브
  • DC모터
  • 스테핑 모터
  • 서보 모터

회로에서는 보통 Y 주소를 사용해서 표현한다. 예: Y0, Y1

공압 장치도 같이 알아야 한다

생산자동화기능사 실기에서는 PLC만 아는 것으로 끝나지 않고, 공압 장치와 센서가 어떻게 연결되는지도 알아야 한다.

Pneumatic

복동 실린더

전진과 후진을 모두 제어할 수 있는 실린더이다.

Valve

솔레노이드 밸브

PLC 출력 신호를 받아 공기의 흐름 방향을 바꾼다.

Sensor

전진/후진 센서

실린더가 끝까지 전진했는지, 후진했는지 확인하는 센서이다.

Air

에어 유니트

공압 시스템에 필요한 압축 공기를 안정적으로 공급한다.

구분 대표 장치 역할 주소 예시
