복동 실린더 제어 회로와 동작 예시

아래 예시는 복동 실린더를 간단하게 제어하기 위한 래더 회로이다. 버튼 1을 누르면 전진하고, 전진 센서가 들어온 뒤 버튼 2를 누르면 후진한다.

간단 래더 회로

입력
X0 : 전진 버튼
X1 : 후진 버튼
X2 : 후진 위치 센서
X3 : 전진 위치 센서

내부 릴레이
M0 : 전진 자기유지
M1 : 후진 자기유지

출력
Y0 : 전진 솔레노이드
Y1 : 후진 솔레노이드


[전진 회로]

 ──┬──[X0]──[X2]──┬──[/X3]────( M0 )
   │              │
   └──────[M0]────┘

 ──[M0]────────────────────────( Y0 )


[후진 회로]

 ──┬──[X1]──[X3]──┬──[/X2]────( M1 )
   │              │
   └──────[M1]────┘

 ──[M1]────────────────────────( Y1 )

동작 순서

  1. 초기 상태에서 실린더는 후진 위치에 있고, 후진 센서 X2가 ON 상태이다.
  2. 버튼 1(X0)을 누르면 X2 조건이 만족되어 전진 회로가 동작하고 M0가 자기유지된다.
  3. Y0 출력이 켜지면서 복동 실린더가 전진한다.
  4. 전진 중에는 전진 센서 X3가 아직 들어오지 않아 버튼 2를 눌러도 후진하지 않는다.
  5. 전진 센서 X3가 ON 된 뒤 버튼 2(X1)를 누르면 후진 회로가 동작한다.
  6. Y1 출력이 켜지면서 실린더가 후진한다.
