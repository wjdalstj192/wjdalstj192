기본 회로 패턴

실기 문제를 풀기 위해서는 회로를 매번 새로 외우는 것이 아니라, 자주 나오는 패턴을 먼저 익히는 것이 중요하다.

스타트 / 스톱 회로

 ──[/X1]──┬──[X0]────────────( Y0 )
          │
          └──[Y0]────────────

버튼 X0를 누르면 Y0가 켜지고, X1을 누르면 꺼진다.

자기유지 회로

 ──┬──[X0]────────────( M0 )
   │
   └──[M0]────────────

한 번 켜진 상태를 계속 유지하기 위해 내부 릴레이를 사용하는 기본 패턴이다.

인터록 회로

 ──[/M1]──[X0]────────( M0 )
 ──[/M0]──[X1]────────( M1 )

두 동작이 동시에 켜지면 안 될 때 서로를 막는 회로이다.

타이머 회로

 ──[X0]───────────────( T0 )
 ──[T0]───────────────( Y0 )

입력이 들어온 뒤 일정 시간이 지나면 출력이 켜지는 회로이다.

카운터 회로

 ──[X0]───────────────( C0 )
 ──[C0]───────────────( Y0 )

입력이 몇 번 들어왔는지 세고, 설정 횟수에 도달하면 출력이 켜지는 회로이다.

실린더 전진/후진 기본 회로

 ──┬──[X0]──[X2]──┬──[/X3]────( M0 )
   │              │
   └──────[M0]────┘

 ──┬──[X1]──[X3]──┬──[/X2]────( M1 )
   │              │
   └──────[M1]────┘

전진과 후진을 조건에 따라 순서대로 제어하는 기본 패턴이다.
