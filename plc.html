PLC Overview

PLC란 무엇인가

PLC는 Programmable Logic Controller의 약자로, 자동화 설비를 제어하기 위한 산업용 제어 장치이다. 쉽게 말하면 자동화 시스템의 두뇌 역할을 하며, 입력을 받아 판단한 뒤 출력 장치를 제어한다.

PLC의 역할

  • 센서로부터 입력 신호를 받는다.
  • 프로그램에 따라 조건을 판단한다.
  • 실린더, 모터, 경광등 같은 출력 장치를 제어한다.
  • 반복 작업을 자동으로 수행한다.

왜 PLC를 사용할까

  • 반복 작업을 자동화할 수 있다.
  • 정확하고 일정한 제어가 가능하다.
  • 산업현장에 적합한 안정성을 가진다.
  • 프로그램 수정으로 제어 변경이 쉽다.

PLC의 기본 동작 원리

입력 → 판단 → 출력의 흐름으로 이해하면 된다.

PLC와 함께 사용되는 장치

Input

센서

물체의 유무, 위치, 거리, 재질 등을 감지하여 PLC에 입력 신호를 보낸다.

Logic

프로그램

PLC가 어떤 조건에서 어떤 동작을 해야 하는지 결정하는 기준이 된다.

Output

실린더

공압을 이용한 직선 운동 장치로, 밀기나 고정 같은 동작에 사용된다.

Output

모터

회전 운동이 필요한 장치에 사용되며, DC모터, 스테핑 모터, 서보 모터 등이 있다.
