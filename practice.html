기본 연습문제

이 페이지는 왕초보가 직접 입력, 출력, 조건을 생각해보는 훈련을 하기 위한 페이지이다. 바로 정답을 외우는 것보다, 먼저 어떤 회로가 필요한지 스스로 생각하는 것이 중요하다.

문제 1

버튼 1을 누르면 램프가 켜지고, 버튼 2를 누르면 램프가 꺼지는 회로를 생각해보자.

  • 입력은 무엇인가?
  • 출력은 무엇인가?
  • 자기유지가 필요한가?

문제 2

센서가 물체를 감지하면 모터가 3초 뒤에 켜지는 회로를 생각해보자.

  • 입력은 무엇인가?
  • 출력은 무엇인가?
  • 타이머가 필요한가?

문제 3

전진 버튼을 누르면 실린더가 전진하고, 전진 센서에 닿으면 멈추는 회로를 생각해보자.

  • 초기 상태는 어디인가?
  • 종료 조건은 무엇인가?
  • 자기유지가 필요한가?

문제 4

전진 중에는 후진이 안 되고, 후진 중에는 전진이 안 되도록 만들려면 어떤 개념이 필요한가?

  • 반대 동작을 막는 회로 이름은?
  • a접점과 b접점 중 어떤 것이 자주 쓰이는가?

풀이 힌트

문제 1 힌트

스타트/스톱 회로와 자기유지를 같이 생각하면 된다.

문제 2 힌트

센서 입력 + 타이머 + 모터 출력의 구조로 나눠서 생각하면 된다.

문제 3 힌트

시작 조건은 버튼, 해제 조건은 전진 센서로 생각하면 쉽다.

문제 4 힌트

인터록 회로 개념이 필요하다.
