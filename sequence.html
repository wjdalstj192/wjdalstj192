시퀀스 제어를 생각하는 방법

생산자동화기능사 실기에서는 회로를 그냥 외우기보다, 동작 순서를 먼저 정리하고 그 순서를 회로로 옮기는 사고가 중요하다.

문제 풀 때 먼저 정리할 것

1. 초기 상태

처음에 실린더나 모터가 어디에 있는지, 어떤 센서가 ON인지 확인한다.

2. 시작 조건

어떤 버튼이나 센서가 동작의 출발점이 되는지 정리한다.

3. 종료 조건

어디까지 가면 멈추는지, 어떤 센서에서 동작이 끝나는지 본다.

4. 금지 조건

동시에 일어나면 안 되는 동작이 있는지 확인하고 인터록을 생각한다.

시퀀스 흐름 예시

복동 실린더의 전진/후진 문제를 예로 들면 이렇게 생각할 수 있다.

초기 상태
후진 위치
→
전진 버튼 입력
→
실린더 전진
→
전진 센서 도달
후진 버튼 입력
→
실린더 후진
→
후진 센서 도달
→
초기 상태 복귀

회로로 바꿀 때 핵심

  • 시작 조건은 a접점으로 잡는다.
  • 멈춰야 하는 조건은 b접점으로 많이 잡는다.
  • 동작 중 유지가 필요하면 자기유지를 넣는다.
  • 반대 동작은 인터록으로 막는다.

초보자가 자주 하는 실수

  • 입력과 출력을 섞어서 생각함
  • 초기 상태를 안 보고 회로를 바로 짬
  • 종료 센서를 안 넣음
  • 전진/후진을 동시에 허용하는 회로를 짬
